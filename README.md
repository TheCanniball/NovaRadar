# Nova Radar

**Next-generation network scanner for Android**

Nova Radar is a powerful and intelligent network scanning tool designed to help you find the fastest and most reliable IP addresses for CDN-based services. Whether you're optimizing your connection or testing network performance, Nova Radar delivers professional-grade results in a beautiful, user-friendly interface.

---

## Features

### 🔍 Smart IP Scanning
- **Dual-mode scanning** – Quick TCP connectivity check + deep TLS handshake verification for maximum accuracy
- **Real-time results** – Watch IPs get verified live with instant feedback
- **Intelligent grading** – Automatic S/A/B/C/D/F rating based on connection quality and latency
- **Colo detection** – Identifies which Cloudflare data center serves each IP

### 📥 Flexible Input Methods
- **Manual IP entry** – Paste or type specific IP addresses for targeted scanning
- **CIDR range scanning** – Scan entire IP ranges automatically
- **Custom ports** – Configure which port to scan (443, 80, 8443, etc.)

### 📊 Powerful Output Options
- **Copy All** – One-tap copy of all working IPs
- **Top 10** – Automatically selects and copies the 10 fastest IPs
- **Pack Greens** – Batch copy all high-quality (Grade A-C) IPs at once
- **Export to TXT** – Save results to a text file in your Downloads folder
- **Nova Proxy compatible** – Optional suffix system for seamless integration

### 🎨 Beautiful & Accessible
- **Dark theme** – Easy on the eyes with a modern, premium design
- **Bilingual support** – Full English and Persian (فارسی) interface with RTL layout
- **Responsive UI** – Optimized for all screen sizes from phones to tablets
- **Material 3 design** – Clean, intuitive, and professional

### ⚙️ Customizable
- **Adjustable threads** – Control scanning speed vs. resource usage
- **Configurable timeout** – Set your preferred connection timeout
- **Custom SNI** – Specify SNI hostname for TLS probes
- **Flexible suffix** – Configure Nova Proxy suffix format to match your setup

---

## Getting Started

1. Download the latest APK from the [Releases](https://github.com/TheCanniball/NovaRadar/releases) page
2. Install on your Android device (Android 7.0+)
3. Enter IPs manually or use CIDR range mode
4. Tap **Start Scan** and watch results appear in real-time
5. Use the **Results** tab to copy or export your findings
